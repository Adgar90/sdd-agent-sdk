# Changelog

## 1.0.0

Initial public version of SDD Agent SDK.

Includes:

- generic `.sdd/` workspace structure;
- `init-sdd.py` installer and validator;
- workspace context schema;
- examples for existing codebase and functional document;
- agent adapters for Codex, Claude Code, Cursor, Kiro and OpenCode;
- operations policies;
- living memory model;
- progressive context loading;
- bilingual documentation in English and Spanish.
