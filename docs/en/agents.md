# Agent Adapters

The SDD core is agent-agnostic.

| Agent | Adapter |
|---|---|
| Codex | `AGENTS.md` |
| Claude Code | `CLAUDE.md` |
| Cursor | `.cursor/rules/sdd-workflow.mdc` |
| Kiro | `.kiro/steering/sdd-workflow.md` |
| OpenCode | `opencode.json` |

Adapters point to `.sdd/`. They do not replace it.
