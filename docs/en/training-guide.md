# Training Guide

## Objectives

Understand:

- SDD concepts;
- issue vs spec;
- `.sdd/` structure;
- workstreams;
- living memory;
- context loading;
- task-by-task implementation;
- external synchronization policy.

## Exercises

1. Identify all core files.
2. Create a discovery spec.
3. Implement or document one task.
4. Review a diff.
5. Update workspace memory.
