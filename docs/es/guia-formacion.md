# Guía de formación

## Objetivos

Entender:

- conceptos SDD;
- issue vs spec;
- estructura `.sdd/`;
- workstreams;
- memoria viva;
- carga de contexto;
- implementación task por task;
- política de sincronización externa.

## Ejercicios

1. Identificar todos los ficheros core.
2. Crear una spec de discovery.
3. Implementar o documentar una task.
4. Revisar un diff.
5. Actualizar memoria del workspace.
