# Herramientas, plugins y MCPs

Las herramientas externas ayudan a leer o sincronizar contexto.

No sustituyen las specs.

Requieren aprobación explícita:

- actualizar issues;
- comentar;
- cambiar estados;
- publicar documentación;
- commit;
- push;
- scripts que modifican ficheros.

Si hay duda entre lectura y escritura, tratarlo como escritura.
