# Memoria viva

La memoria del proyecto debe estar versionada en el repo, no solo en el chat o en la memoria de un agente.

Niveles:

| Nivel | Ubicación |
|---|---|
| Estado macro | `.sdd/WORKSPACE_STATE.md` |
| Memoria de spec | `.sdd/specs/<spec>/notes.md` |
| Decisiones | `.sdd/decisions/` |
| Catálogo/estado | `SDD_INDEX.md` |

Regla:

```text
WORKSPACE_STATE.md = resumen corto
notes.md = detalle de una spec
decisions/ = por qué cambió algo estable
SDD_INDEX.md = mapa de specs y estados
```
