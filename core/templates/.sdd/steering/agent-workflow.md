# Agent Workflow

1. Select issue, brief, request or work item.
2. Locate or create SDD spec.
3. Read `AGENTS.md`, `SDD_INDEX.md` and `.sdd/CONTEXT_LOADING.md`.
4. Read only relevant steering/operations files.
5. Read active spec.
6. Check Git status if files may change.
7. Implement or analyze one task only when instructed.
8. Update `tasks.md` and `notes.md`.
9. Run tests or document why not.
10. Stop for review.
