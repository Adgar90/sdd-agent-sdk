# Memory Policy

| Level | File/Folder | Purpose | Update When |
|---|---|---|---|
| Spec memory | `specs/<spec>/notes.md` | Detailed execution memory | After each task |
| Macro state | `WORKSPACE_STATE.md` | Short current workspace state | When focus/status/workstreams change |
| Decisions | `decisions/` | Historical rationale | When stable context changes |
| Index | `SDD_INDEX.md` | Spec catalog/status/dependencies | When specs/status/deps change |

A workspace may have multiple active workstreams. `WORKSPACE_STATE.md` must not imply that all work belongs to one workstream.

Agent-specific memory is not the project source of truth. The repo SDD memory is the portable source of truth.
