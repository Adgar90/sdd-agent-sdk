# Agent Compatibility

The SDD core is agent-agnostic. The contract is the active spec under `.sdd/specs/`.

| Agent / Tool | Adapter |
|---|---|
| Codex | `AGENTS.md` |
| Claude Code | `CLAUDE.md` |
| Cursor | `.cursor/rules/sdd-workflow.mdc` |
| Kiro | `.kiro/steering/sdd-workflow.md` |
| OpenCode | `AGENTS.md` and optional `opencode.json` |
