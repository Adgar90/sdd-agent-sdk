# New Spec Workflow

1. Read `AGENTS.md`, `SDD_INDEX.md` and `.sdd/CONTEXT_LOADING.md`.
2. Read relevant steering.
3. Read source context.
4. Search existing specs.
5. Classify spec type and workstream relationship.
6. Create `.sdd/specs/<source-id-or-short-name>/`.
7. Copy `.sdd/templates/spec-template/`.
8. Fill all files.
9. Update `SDD_INDEX.md`.
10. Update `WORKSPACE_STATE.md` only if macro state changes.
11. Stop for human review.
